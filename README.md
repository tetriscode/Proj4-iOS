## Build
Add this repository to your projects Cartfile
```
github 'tetriscode/Proj4-iOS' == 4.9.2
```
and run ```carthage update```

## Usage

* Include built proj4.framework into xcode project and add header
```objc
#import <proj4/proj4.h>
```
